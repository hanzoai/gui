# `create-hanzo-gui`

```sh
npm create gui
```

A script that creates a [gui monorepo](https://github.com/hanzoai/gui/tree/main/example-monorepos/blank) for you in seconds.

## Contributing

- Clone the root repo.
- `cd packages/create-hanzo-gui`
- `yarn`
- `yarn test` to build your local app into a gitignored `create-test-app` folder
