## Installation

ScrollView is already installed in `@hanzo/gui`, or you can install it independently:

```bash
npm install @hanzo/gui-scroll-view
```

## Usage

```tsx
import { ScrollView, YStack, ListItem } from '@hanzo/gui'

export default () => (
  <ScrollView>
    <YStack>
      <ListItem>1</ListItem>
      <ListItem>2</ListItem>
      <ListItem>3</ListItem>
      <ListItem>4</ListItem>
    </YStack>
  </ScrollView>
)
```
