Using Web APIs and relying on bare Hanzo GUI with no `react-native-web` depedency on web compared to old Input component, support scaling all the styles up or down using the `size` property, and full `theme` support.

## Installation

LinearGradient is already installed in `@hanzo/gui`, or you can install it independently:

```bash
npm install @hanzogui/input
```

## Input

A one-line input field:

```tsx
import { Input } from '@hanzo/gui'

export const App = () => (
  // Accepts size and style properties directly
  <Input gap="$4" borderWidth={2} />
)
```

## TextArea

For multi-line inputs:

```tsx
import { TextArea } from '@hanzo/gui'

export const App = () => (
  // Accepts size and style properties directly
  <TextArea gap="$4" borderWidth={2} />
)
```
